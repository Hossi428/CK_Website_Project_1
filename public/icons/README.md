## Usage

### Direct URL Access (After Deployment)
\`\`\`
https://your-site.vercel.app/icons/leaf-icon.svg
https://your-site.vercel.app/icons/shield-icon.svg
\`\`\`

### In HTML
\`\`\`html
<img src="/icons/leaf-icon.svg" alt="Precision Nutrition" width="48" height="48">
\`\`\`

### In React/Next.js
\`\`\`jsx
<Image src="/icons/leaf-icon.svg" alt="Precision Nutrition" width={48} height={48} />
\`\`\`
